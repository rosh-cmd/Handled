## Fonts (optional)
Place Glook webfont files in `assets/fonts/`:
- `fonts/Glook-Bold.woff2`
- `fonts/Glook-Bold.woff`

Then uncomment the @font-face block in style.css.

## Tips
- JPG is fine for photos. PNG for logos/transparency.
- Aim for ~150–300 KB per image for fast loading.
- Images can be exported directly from Wix via the Media Manager.
